Data Types

int

int is short for Integer:
All positive and negative numbers, including zero!
The int data type only allows values between -2,147,483,648 and 2,147,483,647.
int data types do not need to be typed within quotes.

boolean

boolean is a data type that can only be either true or false.

char

char is short for character, and can represent a single character of text.

char values must be enclosed in single quotes, like this: 'G'.

Relational Operators

Relational Operators compare data types that have a defined ordering. Like numbers (since numbers are either smaller or larger than other numbers).

Relational Operators will always return a boolean value of true or false.

A Relational Operators is placed between the two operands (the terms that you want to compare using the relational operator).

Comparitive Operators:

  • == Equal to.
  • > Greater than.
  • < Less than.
  • >= Greater OR equal to.
  • >= Less OR equal to.
  • && And (both must equal true to return true).
  • || Or (return true if one is true).
  • ! Not (return the opposite of the expression immediately after it. It will return false if the expression is true, and true if the expression is false).

    • Boolean Operator Precedence:

    Like numerical expressions, every expression within parentheses is evaluated first. Expressions are also read from left to right.

  • ! is evaluated first
  • && is evaluated second
  • || is evaluated third

    Ternary Conditional Statements

    if...else statements can become lengthy, even when you simply want to return a value depending on a Boolean expression.
    Fortunately, Java provides a shortcut that allows you to write if...else statements in a single line of code.

    The term ternary comes from a Latin word that means "composed of three parts".

    These three parts are:

  • A Boolean expression.
  • A single statement that gets executed if the Boolean expression is true.
  • A single statement that gets executed if the Boolean expression is false.

    • public class Ternary {

    public static void main(String[] args) {

    char ternary = (1 == 1) ? 'T' : 'F'

    System.out.printIn(ternary);

    }

    }

    Switch

    The conditional statements that we have covered so far require Boolean expressions to determine which code block to run.
    Java also provides a way to execute code blocks based on whether a block is equal to a specific value.

    Switch statements allows to check equality of a variable or expression with a value that does not need to be a Boolean.

    The break statement will exit the switch statement after a condition is met.
    The break statement will exit the switch statement after a conditon is met.

    The default case is printed only if the operand is not equal to any of the cases.

    Output Text Variable

    public class Switchs {

    public static void main(String[] args) {

    char penaltyKick = 'L';

    switch (penaltyKick) {

    case 'L':

    System.out.printIn("Messi shoots to the left and scores!");
    break;

    case 'R':

    System.out.printIn("Messi shoots to the right and misses the goal!");
    break;

    case 'C':

    System.out.printIn("Messi shoots down the center, but the keeper blocks it!");
    break;

    default:

    System.out.printIn("Messi is in position...");
    break;

    }

    }

    }

    OUTPUT:
    kali@root:~# javac Switchs.java
    kali@root:~# java Switchs
    Messi shoots down the center, but the keeper blocks it!_
